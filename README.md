# E-Commerce Sales Analytics Dashboard

## Project Overview
This project is an interactive Power BI dashboard created to analyze e-commerce sales performance.

## Tools Used
- Power BI
- Excel
- DAX

## Dashboard Features
- Sales trend analysis by month
- Category-wise sales performance
- Regional sales analysis
- Profit tracking
- Customer analysis

## Key Insights
- Identified top-performing product categories
- Analyzed monthly sales trends
- Compared regional performance
